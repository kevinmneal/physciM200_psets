Paper questions

These errors can all lead to inappropriate conclusions from the data by using datasets that violate the assumptions of models, when better models are available. Can lead to false positives or false negatives in assessing statistical independence.

  1. Failing to state number of cases used in correlation; bad because sample size impacts the strength of correlation
  2. No 95% CI with correlation coefficient
  3. Inappropriate use of a regression line in a scatterplot through an array when only a correlation is involved (i.e. linear regression through cloud of points)
  4. Failing to explain/justify inclusion of outliers in a plot and computations; can lead to inappropriate conclusions about correlations of data based on just a few points
  5. Use of correlation coefficient when sample size is small or with a non-linear trend
  6. Using Pearson on non-Gaussian, heteroscedastic, or non-linear data
  7. Not reporting heteroscedasticity of data
  8. Inflating importance of a significant outcome based on correlation alone
  9. Failure to report number of data points
  10. Using regression and correlation interchangeably
  11. Justifiably using a scatterplot without detailing the correlation coefficient given
  12. Failing to state type of regression line (LSR, MA, RMA)
  13. Failure to draw hyperbolic confidence interval lines around regression line, given an appropriate regression line
  14. Parallel CI lines, or one parallel and one hyperbolic CI line, around a regression line (CI lines are hyperbolic around regression lines, reflecting greater uncertainty at extremes of the sample distribution)
  15. Comparing slopes of two undefined regression lines by [visual?] inspection rather than by an analysis of covariance statistic

Bootstrapping pairwise data to get effect size with CI

bootind <- function(datasetA, datasetB, reps=10000) {
  indices = c(1:length(datasetA))
  mediandifference = rep(NA,reps)
  for (i in 1:reps) {
    bootindices = sample(indices,length(indices),replace=TRUE)
    bootA = datasetA[bootindices]
    bootB = datasetB[bootindices]
    bootdiff = bootB - bootA # notice this is B - A
    mediandifference[i] = median(bootdiff)
    }
  invisible(mediandifference)
}
#result of this will be the change from timepoint A to timepoint B; if negative, means there was a decrease in measure from A to B

bootind.nhst <- function(datasetA, datasetB, reps=10000) {
  medianbootdiff = rep(NA,reps)
  for (i in 1:reps) {
    signs = sample(c(-1,1),length(datasetA), replace=T)
    difference = datasetB - datasetA
    bootdifference = signs * difference
    medianbootdiff[i]= median(bootdifference)
  }
  invisible(medianbootdiff)
}

booter2 <- function(dataA, dataB=NULL, reps=10000, stat=NULL, ...) {
  bootstat <- rep(NA, reps)
  if (!is.null(dataB)) {
    indices <- c(1:length(dataA))
    for (i in 1:reps) {
      bootindices <- sample(indices, length(indices), replace=T)
      bootstat[i] <- stat(dataA[bootindices], dataB[bootindices])
    }
  } else {
    for (i in 1:reps) {
      bootdata <- sample(dataA, length(dataA), replace=TRUE)
      bootstat[i] <- stat(bootdata)
    }
  }
  invisible(bootstat)
}



spearman <- function(dataA, dataB) {
  cor(dataA, dataB, method="spearman")
}

# sample without replacement; shuffle pairs for A and B
bootcor.nh <- function(dataA, dataB=NULL, reps=10000, stat=NULL, ...) {
  bootstat <- rep(NA, reps)
  if (!is.null(dataB)) {
    indices <- c(1:length(dataA))
    for (i in 1:reps) {
      bootindices <- sample(indices, length(indices), replace=F)
      bootstat[i] <- stat(dataA[bootindices], dataB)
    }
  } else {
    for (i in 1:reps) {
      bootdata <- sample(dataA, length(dataA), replace=TRUE)
      bootstat[i] <- stat(bootdata)
    }
  }
  cat("pvalue (upper tail):", length(bootstat[bootstat>cor(dataA, dataB, method="spearman")])/reps)
  invisible(bootstat)
}

cor1.nh <- bootcor.nh(IL.1b, IL.10, stat=spearman) # p = 0.0952
## pvalue (upper tail): 0.0877

## pvalue (upper tail): 0
